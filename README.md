A premium, offline-capable music player for iOS (iPad/iPhone) and modern browsers. Built with Vanilla JS, HTML5 Audio, and IndexedDB.

## Features
- **Offline Playback**: Works without internet once installed.
- **Speed Control**: Adjust playback speed from 0.5x to 1.5x.
- **Pitch Control**: Toggle pitch preservation (Time Stretch vs. Varispeed).
- **Persistent Library**: Songs are saved in the browser's IndexedDB.
- **PWA Support**: Installable on iOS Home Screen.

## How to Deploy (GitHub Pages)
1.  Upload these files to a GitHub repository.
2.  Go to **Settings** > **Pages**.
3.  Under **Source**, select `Deploy from a branch`.
4.  Select `main` (or `master`) branch and `/ (root)` folder.
5.  Click **Save**.
6.  Visit the generated URL on your iPhone/iPad.

## How to Install (iOS)
1.  Open the GitHub Pages URL in **Safari**.
2.  Tap the **Share** button.
3.  Select **Add to Home Screen**.
4.  Launch the app from the icon.
